A gesture-controlled virtual drawing application built using Python, OpenCV, and MediaPipe. Users can draw, resize, and fill shapes in the air using hand gestures, without touching the scree.

Features:
1) Hand-gesture based drawing using webcam
2) 12 color options and 12 basic shapes
3) Live shape preview with resize support
4) Fill or outline shapes
5) Undo functionality
6) Adjustable brush size
7) Minimal single-line UI

Controls:
1) Index finger – Draw
2) Index + Middle – Select color / shape
3) Thumb – Open menus
4) + / - – Brush size
5) F – Toggle fill
6) U – Undo
7) C – Clear
8) S – Save
9) Q – Quit

Use Cases:
> Computer vision mini project.
> Gesture-based UI demo.
> Learning OpenCV & MediaPipe.
